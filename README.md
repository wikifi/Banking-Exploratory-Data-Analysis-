# Banking-Exploratory-Data-Analysis-

## Overview

This project focuses on analyzing banking customer data using **Python, SQL, and Data Analytics techniques** to uncover meaningful business insights. The notebook includes data extraction, preprocessing, exploratory data analysis (EDA), and visualization to understand customer behavior, income distribution, loyalty classification, and banking trends.

The project demonstrates practical applications of data analytics and visualization for decision-making and customer analysis in the banking domain.

---

# Objectives

* Analyze banking customer data for trends and insights
* Perform data cleaning and preprocessing
* Conduct exploratory data analysis (EDA)
* Visualize customer demographics and financial patterns
* Generate business insights using statistical analysis

---

# Features

* Data extraction from MySQL database
* Data preprocessing and transformation
* Descriptive statistical analysis
* Univariate and bivariate analysis
* Interactive and informative visualizations
* Income segmentation and customer categorization

1. Install required libraries:


2. Open the notebook in Jupyter Notebook or VS Code.

3. Configure MySQL database credentials if required.

4. Run the notebook cells sequentially.
